# str8.js
> A simple string manipulation library written in Javascript.

## Use
```javascript
// load the module
var str8 = require('str8.js');

// ask for a number's ceil
str8.firstHalf("test");
```

## Test
To execute the library's tests, first install the project dependencies:

```
$ npm install
```

Then, run the tests
```
$ npm test
```
